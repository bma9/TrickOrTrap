To play this game you must move between locations typing one word commands.

Some commands are motion commands.  These will move you from room to room.
Available commands are provided above the input text box at the bottom of the game screen.
Some commands include: LEFT, RIGHT, FORWARD, BACK, INSPECT, EXIT, NEXT, SKIP

Not all motions are possible in every room. In addition, some rooms may have "special" or "secret" motion commands.

There are other action commands in the game. These include:

HELP: this will display instructions
INVENTORY: this will print your current inventory.
OBSERVE: this will print the description for the current room.
SAVE: this will quicksave the game with an untitled save slot.

Some shortcut inputs for the commands are:
L - LEFT
R - RIGHT
F - FORWARD
B - BACK
I - INSPECT
E - EXIT
S - SAVE
O - OBSERVE
H - HELP
N - NEXT
INV - INVENTORY

Some paths may be blocked.  To unblock a path you may need a specific object to be in your inventory.

The game is over when your player reaches the VICTORY room, or when your player DIES.
